## ✅ Key Features
- Two-column Hero with profile photo, floating animated badges, spinning rings
- About, Skills, Experience Timeline, Projects, Contact sections
- SEO meta tags (OG, author, keywords)
- Fully responsive — mobile hamburger menu included
- Custom cursor with ring-follow animation
- Scroll reveal animations
- Contact form with validation
- No external frameworks — pure HTML/CSS/JS

## 🛠️ Tech Stack
| Layer | Technology |
|-------|-----------|
| Frontend | HTML5, CSS3, Vanilla JavaScript |
| Fonts | Google Fonts (Space Mono + Syne) |
| Deployment | GitHub Pages |
